## üá¨üáß Project Overview

This repository contains code and exported analysis artifacts developed as part of a research project on cardiovascular adaptation in athletes with disabilities in the presence of connective tissue dysplasia (CTD).

The study focuses on structural and functional cardiovascular markers, including left ventricular false tendons (LV false tendons), and their association with physiological adaptation and functional performance indicators.

The project emphasizes:
- careful preprocessing of medical and physiological data;
- descriptive statistics and hypothesis-driven analysis;
- interpretability of results in a clinical and physiological context;
- reproducibility of the analysis without hosting raw patient-level data.
